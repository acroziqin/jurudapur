``` php artisan migrate:refresh --seed ```

``` php artisan storage:link ```

## Requirement
1. NodeJS
2. NPM

### Installation

Install the dependencies (gulp, gulp-sass, browser-sync)

```sh
$ npm install
```

### Run

This will watch your sass files, compile them and run your dev server at http://localhost:3000

```sh
$ npm start
```
